Yes! With a few small changes, you can have a Customer Voice button point wherever you want. If you open a template in Customer Voice and right-click on the button to highlight it, you can use the "Insert/Edit Link" button to edit its properties.

![Right-click on button to edit properties](/img/customer-voice/faqs/right-click-button.png)

You can change the text that is displayed on the button, as well as where the button leads when clicking it. If you update the "URL" field, you change the button destination. The "Text to Display" changes the text written on the button itself.

![Edit button properties dialog](/img/customer-voice/faqs/edit-link-dialog.png)
