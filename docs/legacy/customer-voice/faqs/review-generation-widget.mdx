:::caution Note
As of February 21st, 2025, Customer Voice has become a legacy Vendasta product. Start using [Reputation Management Premium](https://partners.vendasta.com/marketplace/products/RM) to automatically collect reviews and NPS via email and SMS.
:::

You can embed our Review Generation Widget onto any website. To get the applicable code:

1. Launch **Customer Voice** > Go to **Tools** > **Widgets**.
2. In the Widget Setup section, customize your widget so it matches your branding or website.
3. Once ready, select **Copy Widget Code**.
4. You or your webmaster can paste the Widget Code into the HTML of the desired website.

If you have a WordPress website, please see [this article](/legacy/legacy-customer-voice/legacy-customer-voice-faqs/wordpress-widget).
