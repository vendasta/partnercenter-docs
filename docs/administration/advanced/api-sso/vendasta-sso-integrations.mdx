# Does Vendasta have an option for Single Sign-On (SSO) integrations?

For Partners on a current subscription of Premium or Custom, we offer Single Sign-On (SSO) integrations that enable clients to access products using your system's logins. Detailed SSO documentation can be found at [http://www.vendasta.com/documentation](http://www.vendasta.com/documentation).
