---
title: Do I have access to cPanel if I host my websites with Website Pro?
description: Learn about server administration and hosting options in Website Pro
sidebar_label: Access to cPanel with Website Pro
---

**No.** [cPanel](https://cpanel.net/) is essentially a separate web hosting solution, which is not compatible with Website Pro architecture.

Standard server administration processes, such as domain/certificate management, backup, restore, etc., are fully managed in Website Pro. Users who have more technical experiences may also access site databases/files directly through phpMyAdmin/SFTP on our platform.
