# How does Website Pro handle WordPress core updates

One of the most important things you can do to keep your clients' sites safe from malicious activity is to make sure your WordPress core is up-to-date. That's why Website Pro takes care of this automatically.

Website Pro handles all minor WordPress updates for you. By default, automatic plugin updates are disabled to avoid the crash of the website in case of compatibility issues however you can turn them on with a click of a button.
