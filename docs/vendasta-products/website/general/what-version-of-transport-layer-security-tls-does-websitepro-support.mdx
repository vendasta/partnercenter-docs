# What version of Transport Layer Security (TLS) does WebsitePro support?

**Transport Layer Security** **(TLS)** is a cryptographic protocol designed to provide communications security over a computer network and Website Pro supports versions **1.1**, **1.2** and **1.3** of the protocol.
