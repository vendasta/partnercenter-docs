**To set up WooCommerce payments:**

1) Navigate to the WordPress Dashboard from Website Standard/Pro.

2) Click on "WooCommerce" in the left navigation, then select "Settings."

3) Click on "Payments."

![WooCommerce Settings Navigation](/img/woocommerce-payments/woocommerce-payments-1.jpg)

4) Select the payment method and complete the setup for the selected method.

![WooCommerce Payment Methods](/img/woocommerce-payments/woocommerce-payments-2.jpg)

5) Click save at the bottom of the page.

6) (Optional) To test the payment flow, enable the Sandbox mode.

![WooCommerce Sandbox Mode](/img/woocommerce-payments/woocommerce-payments-3.jpg)
