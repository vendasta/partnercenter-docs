## How to add custom links to SMS review requests

1. Go to Customer Voice > **Templates** > **Add New Template**
2. Select **New SMS Template** > name the template > select **New Custom Site template**
3. Customize the template with your custom link and message ***(Note: clicks on custom links are not tracked)*** > **Save**

![Screenshot showing SMS template customization](/img/legacy-customer-voice/custom-links-sms.jpg)

## Who has access to this?

All **Customer Voice Pro accounts that have an SMS add-on active** will be able to use this feature.

## Businesses can send an SMS message to their customers with a link to any website they wish. This could include, but is not limited to:

- Link to receive a coupon code 
- Link to schedule an appointment/appointment confirmation page
- Link to the company website (i.e. to promote a sale)
