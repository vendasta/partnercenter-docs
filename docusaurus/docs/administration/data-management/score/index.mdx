# Score

This section provides information about configuring and managing scoring systems within Vendasta's data management platform.

## Scoring Configuration

- Lead Scoring - Set up automated scoring for incoming leads
- Opportunity Scoring - Configure scoring criteria for sales opportunities
- Custom Scoring Rules - Create custom rules and weightings for your scoring system
- Score Analytics - View and analyze scoring performance and trends
- Score Actions - Set up automated actions based on score thresholds
