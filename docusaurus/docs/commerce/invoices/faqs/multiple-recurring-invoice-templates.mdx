---
title: Can I set up multiple recurring invoice templates on the same account?
description: Learn about setting up multiple recurring invoice templates on the same account in Vendasta.
sidebar_label: Multiple Recurring Invoice Templates
---

:::caution Discontinued Feature
The Recurring Invoice functionality has been discontinued for new Partners. Please familiarize yourself with the enhanced Subscription Billing feature, which has been introduced as a replacement.
:::

Yes, you can set up as many recurring invoice templates as you want on the same account. Each can have its own products, renewal schedule, payment method, recipient, memo description, etc. by adjusting the date and frequency.
