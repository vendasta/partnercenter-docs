title: How do I create my Proposal template?
description: Learn how to create a custom Proposal template in Vendasta's Commerce module to better present your offerings to clients.
sidebar_label: Creating Proposal Templates
---

To create a Proposal template:

1. In **Partner Center > Commerce**, go to **Proposals > Templates**.
2. Click the **Create New Template** button to start creating your template.
