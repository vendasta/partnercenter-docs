# My Listing Page Event Analytics

Each time a user visiting your listing takes an action, such as viewing the map, we track the event. Events are then added to the Event Analytics chart so businesses can track how engaged their users are with My Listing.

We will track the event whenever a user performs any of the actions listed below.

- View Hours
- View Map
- View Contact Information
- View Video
- Call Business
- Link to Full Website
- View Image
- Scanned QR Code
