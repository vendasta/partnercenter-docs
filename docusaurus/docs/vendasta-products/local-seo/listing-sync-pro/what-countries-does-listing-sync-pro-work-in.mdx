---
title: What countries does Listing Sync Pro work in?
description: Learn about the countries where Listing Sync Pro and Listing Sync Pro powered by Yext are available.
---

Listing Sync Pro is currently available in the US, Canada, Australia, Great Britain, France, Germany, and Italy. Listing Sync Pro powered by Yext is only available in the US and Canada. If you'd like to see another country on the list, please send an email to support@vendasta.com.

For a list of all sources that each country currently syndicates to, please see [this article](/vendasta-products/vendasta-products-local-seo/vendasta-products-local-seo-listing-sync-pro/what-sources-does-listing-sync-pro-syndicate-to).
