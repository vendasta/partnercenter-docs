# Add a user to an account

To add a user to an account:

1. Go to **Partner Center** > **Accounts** > **Manage [Accounts](https://partners.vendasta.com/manage-accounts)**.  
2. Find the account you want to add the user to. If there are no users assigned to the account, you can click the **Add User** icon ![Add User icon](/img/accounts/accounts-manage-users/add-user-icon.png) next to the account name.
   :::note
   You can also click on the account name, scroll to the **Users** section, then click **Add Users**.
   :::
3. Select the user to add to the account.
   a. If the user already exists:
      i. Find the user in the list.
      ii. Click the checkbox next to the user. 
      iii. Click **Add User**. 
   b. If the user doesn't exist:
      i. Click **Create New User**.
      ii. Complete the form with the user's information:
         - First name
         - Last name
         - Email
         - Phone
         - Welcome message
           - Select whether you want the platform to send a welcome message to the user.
             :::note
             If **Send Welcome Message** is selected, the platform will send a welcome email to the user. This email contains a link to log in to Business App. Upon clicking this link, the user can set their password.
             :::
           - Enter a custom message for the top of the welcome email (optional).
             ![Welcome message screen](/img/accounts/accounts-manage-users/welcome-message.png)
      iii. Click **Create User**.

The user will now be assigned to the account.
