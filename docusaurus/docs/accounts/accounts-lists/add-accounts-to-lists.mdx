- [Multiple accounts](#add-multiple-accounts-to-lists)

## Add Individual Accounts to Lists

To add an individual account to a List:

1. Go to **Partner Center** > **Accounts** > **Manage Accounts**.
2. Find the account you want to add to the list. Click the **menu** icon ![Menu icon](/img/accounts/lists/menu-icon.jpg) at the end of the row.
3. Click **Add to List**.
4. Add the account to a List:
   a. If you want to add the accounts to an existing List:
      i. Select the existing List from the dropdown menu.
      ii. Click **Add**.
   b. If you want to add the accounts to a new list:
      i. Go to the **Create New List** tab.
      ii. Enter a name for the List.
      iii. Click **Create and Add**.

## Add Multiple Accounts to Lists

To add multiple accounts to a List:

1. Go to **Partner Center** > **Accounts** > **Manage Accounts**.
2. Filter and search for the accounts you want to add to a List.
   :::note
   You can use filters and the search bar to find your accounts.
   :::
3. Click **Add # to List**.
4. Add the accounts to a List:
   a. If you want to add the accounts to a New List:
      i. Enter a name for the List.
      ii. Click **Create and Add**.
   b. If you want to add the accounts to an existing List:
      i. Go to the **Add to List** tab.
      ii. Select the existing List from the dropdown menu.
      iii. Click **Add**.
