# Add Accounts to Lists 

You can add individual accounts or multiple accounts to Lists. The best way to test this is to make a bunch of changes and see if they stick. We will be changing all of this text anyway.

- [Individual accounts](#add-individual-accounts-to-lists)
- [Multiple accounts](#add-multiple-accounts-to-lists)

## Add Individual Accounts to Lists

To add an individual account to a List:

1. Go to **Partner Center** > **Accounts** > **Manage Accounts**.
2. Find the account you want to add to the list. Click the **menu** icon ![Menu icon](./img/accounts/lists/menu-icon.jpg) at the end of the row.
3. Click **Add to List**.
4. Add the account to a List:
   a. If you want to add the accounts to an existing List:
      i. Select the existing List from the dropdown menu.
      ii. Click **Add**.
   b. If you want to add the accounts to a new list:
      i. Go to the **Create New List** tab.
      ii. Enter a name for the List.
      iii. Click **Create and Add**.

## Add Multiple Accounts to Lists

To add multiple accounts to a List:

1. Go to **Partner Center** > **Accounts** > **Manage Accounts**.
2. Filter and search for the accounts you want to add to a List.
   :::note
   You can use filters and the search bar to find your accounts.
   :::
3. Click **Add # to List**.
4. Add the accounts to a List:
   a. If you want to add the accounts to a New List:
      i. Enter a name for the List.
      ii. Click **Create and Add**.
   b. If you want to add the accounts to an existing List:
      i. Go to the **Add to List** tab.
      ii. Select the existing List from the dropdown menu.
      iii. Click **Add**.

### FAQs

<details>
<summary>Can I add the same account to multiple lists?</summary>

Yes. You can add an account to as many lists as you need. This is useful when you want to organize accounts in different ways - for example, having an account in both a "Newsletter Subscribers" list and a "Premium Customers" list.
</details>

<details>
<summary>What happens if I try to add an account that's already in the list?</summary>

The system will prevent duplicate entries. If you try to add an account that's already in a list, the account won't be added again, and no error message will be displayed.
</details>

<details>
<summary>Is there a limit to how many accounts I can add to a list at once?</summary>

When using the bulk add feature, you can add any number of filtered accounts to a list. However, for optimal performance, it's recommended to work with batches of up to 1000 accounts at a time when performing bulk operations.
</details>
