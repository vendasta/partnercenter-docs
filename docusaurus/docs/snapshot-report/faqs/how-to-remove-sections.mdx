---
title: How do I remove sections of the Snapshot Report?
sidebar_label: Removing sections
description: Learn how to remove sections from the Snapshot Report for all accounts or specific markets.
---

You can remove default Snapshot Sections for **all accounts** by navigating to **Partner Center** > **Customize** > **Partner defaults** > **Sales** > **Edit Default Snapshot Template** > Toggle off the section you wish to remove.

![Screenshot showing how to remove sections from the Snapshot Report for all accounts](/img/snapshot-report-faqs/remove-sections-1.png)

![Additional screenshot showing the toggle option to remove sections](/img/snapshot-report-faqs/remove-sections-2.png)

You can remove default Snapshot Sections for each Market via **Partner Center** > **Customize** > **Markets** tab > Select **[Market Name]** > **Sales** > **Edit Default Snapshot Template** > Check the box of the section you wish to remove.
